Employment Quiz CDN

This page hosts the employment law quiz JavaScript file for Webflow integration.

Available Files

webflow-quiz.min.js 29.9 KB
quiz-bypass-orb.js 1.5 KB

Usage in Webflow

Add this code to your Webflow HTML Embed element:

<div id="employment-quiz"></div>
<script src="https://claimquiz.andy-b1f.workers.dev/webflow-quiz.min.js" defer></script>

ORB Bypass Solution

If you encounter ORB blocking, use this instead:

<div id="employment-quiz"></div>
<script>
(async function() {
  try {
    const response = await fetch('https://claimquiz.andy-b1f.workers.dev/webflow-quiz.min.js');
    const scriptContent = await response.text();
    const script = document.createElement('script');
    script.textContent = scriptContent;
    document.head.appendChild(script);
  } catch (error) {
    console.error('Quiz loading failed:', error);
  }
})();
</script>
